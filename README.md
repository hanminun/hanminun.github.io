# 🌊 《[海边的许愿瓶](https://hanminun.github.io/)》

阿凯把第十三个许愿瓶扔进海里时，海浪正拍打着礁石，溅起的水花打湿了他的裤腿。透明的玻璃瓶在蓝色的海面上晃了晃，然后被浪卷着，慢慢漂向远方。

他和小夏是在海边的救生队认识的，她是志愿者，负责给游客讲解安全知识，扎着高高的马尾辫，笑起来像夏天的太阳。他是救生员，每天穿着橙色的救生衣在沙滩上巡逻，看见她给小朋友系救生圈时，会故意吹一声响亮的口哨，惹得她回头瞪他一眼，却忍不住笑出声。

他们会在傍晚时沿着海岸线散步，她捡形状奇怪的贝壳，他帮她提着装满贝壳的塑料袋。她会把贝壳贴在耳边，听海浪的声音，然后说“大海在唱歌”。他会在她不注意时，把一只小螃蟹放在她的手心，看着她尖叫着跳起来，然后笑着把螃蟹放回海里。

去年夏天台风来袭，阿凯在营救一个被困的游客时，被海浪卷到了礁石上，腿部受了重伤。小夏在医院里守了他三天三夜，眼睛熬得通红。他醒来时，看见她趴在床边，手里握着一个用贝壳串成的手链，上面刻着他们名字的缩写。

阿凯出院后就退役了，在海边开了一家小酒馆。每天傍晚，小夏会坐在酒馆门口，帮他招呼客人，夕阳把她的影子拉得很长，像一幅温暖的画。他们会在打烊后坐在沙滩上，把写着愿望的纸条塞进许愿瓶，然后一起扔进海里。

昨天我去他们的酒馆喝酒，阿凯正在吧台后面调酒，小夏坐在旁边，把剥好的花生一颗一颗放进他嘴里。海浪拍打着沙滩，发出有节奏的声响，酒馆里的灯光很暖，映着他们相视而笑的脸，像一杯加了糖的鸡尾酒，甜得让人沉醉。

金句： 爱情就像大海，有时平静，有时汹涌，但只要你在身边，我就有勇气驶向任何远方。
